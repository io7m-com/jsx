2.1.1
Unicode
1
The specification makes reference to the Unicode character set which, at the time of writing, is at version 6.2.0. The specification often references specific Unicode characters, and does so using the standard notation U+NNNN, where N represents a hexadecimal digit. For example, U+03BB corresponds to the lowercase lambda symbol Î».
2.1.2
EBNF
1
The specification gives grammar definitions in ISO/IEC 14977:1996 Extended Backus-Naur form.
